* Personal dotfiles

** Configs

Example usage:

#+begin_src shell
  cd dotfiles/
  # Dry run
  stow -n -v -t ~ emacs
  # Deploy
  stow -t ~ emacs # add .emacs.d/ to ~
  stow -t ~/.config niri

  # remove symlink
  stow -t ~/.config -D niri
#+end_src

+ Skip =-t= if layout is =~/dotfiles=.
+ Use =--adopt= to gather files from current config. This is from *target* to *source*.

** Portages

Use script to diff the config. Do not symlink configs.
